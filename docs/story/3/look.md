# 查看

数值游戏离不开数值，数值离不开随机性，也应该有多种多样的查看方法。

最基本的查看方法就是直接用`查看`。

此时，面粉团就会告诉你，你的个人信息。包括境界、修为、三相、物品。

也有10%概率同时中二的念一遍你所在境界的slogan。

## 查看表

| 命令          | 反馈                     |
| ------------- | ------------------------ |
| `看大佬`      | 展示玩家修为排行榜 |
| `查看#物品名` | 查看一个物品的描述       |
| `商店`        | 进入商店                 |
| `菜单`        | 显示大部分命令           |
| `交易所`      | 进入珠宝交易所           |
| `成就榜`      | 查看所有被实现的成就           |
| `查看#怪物名`      | 查看一个怪物的描述           |
| `遗迹`      | 查看当前位于的遗迹           |
